# CatCal - 你的猫咪营养师 🐱

AI 驱动的饮食营养记录 App。拍照或文字记录每一餐，猫咪助手帮你分析热量和营养。

## 核心功能

- 📸 **拍照识别** — AI 自动识别食物种类，估算热量和营养成分
- ⌨️ **文字记录** — 输入食物描述，智能换算蛋白质、脂肪、碳水
- 📊 **营养分享** — 生成精美营养图文卡片，一键分享社交平台
- 📅 **饮食历史** — 按时间筛选，自动汇总卡路里摄入
- 🚶 **运动追踪** — 同步 HealthKit 步数，自动设定运动目标
- 🐱 **猫咪陪伴** — 可爱的猫咪助手全程陪伴

## 技术亮点

- 集成中国食物成分表（第 6 版）5,688 条真实营养数据（RAG）
- Apple 原生登录，iCloud Private Database 存储
- SwiftUI + UIKit 混合架构
